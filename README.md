# Ghidra Catppuccin Mocha Theme

A dark theme for [Ghidra](https://ghidra-sre.org/) based on the [Catppuccin Mocha](https://github.com/catppuccin/catppuccin) color palette.

Designed to match a Neovim Catppuccin Mocha setup — same syntax colors, same background tones, same vibe.

## Install

1. Copy `Catppuccin_Mocha.theme` to your Ghidra themes directory:
   - **macOS**: `~/Library/ghidra/<version>/themes/`
   - **Linux**: `~/.ghidra/<version>/themes/`
   - **Windows**: `%USERPROFILE%\.ghidra\<version>\themes\`

2. In Ghidra, go to **Edit > Theme > Change...** and select **Catppuccin Mocha**.

## Compatibility

Built for Ghidra 12.0.4. Should work with any version that supports the Flat Dark Look and Feel (11.0+).
